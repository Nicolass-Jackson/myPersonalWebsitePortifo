# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website!  
This project showcases my skills, projects, and experience as a full stank web developer/designer.  
It serves as both a professional profile and a creative space to share my work.

---

## 🚀 Features

- 💼 **About Me** section — introduces who I am and what I do.  
- 🧠 **Skills** — highlights my technical and professional expertise.  
- 🖥️ **Projects** — displays selected works with descriptions, technologies used, and live/demo links.  
- 📬 **Contact Form** — allows visitors to send messages directly through the website.  
- 🌙 **Responsive Design** — fully optimized for desktop, tablet, and mobile devices.  
- 🎨 **Modern UI** — clean layout with smooth animations and consistent branding.
